  1. Open your OhSheet spreadsheet
  2. Click Extensions → Apps Script
  3. Delete all existing code, paste the script, click Save (💾)
  4. Reload your spreadsheet — an OhSheet menu appears
  5. Click any sync option — Google will ask for permission once

Seeing "app not verified"? This is normal for private scripts. Click Advanced → Go to OhSheet (unsafe) to continue. The script only needs permission to add a menu to this spreadsheet and call the sync API.
